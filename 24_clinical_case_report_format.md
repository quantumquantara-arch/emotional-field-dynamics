# Purpose of This Format

This case-report framework ensures that EFD work is:
- rigorous  
- ethical  
- non-coercive  
- trauma-informed  
- grounded in dynamic systems  
- free from diagnostic labels  
- centered on field movement  
- aligned with EFD principles  
- clinically useful  
- research-ready  

It also establishes a consistent standard across training, supervision, publications, and certification.
